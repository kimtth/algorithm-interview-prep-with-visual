Given a 1-indexed sorted array, find two numbers that add up to target. Return their indices.

Key Insight: Use bisect_left's lo parameter to specify search start, avoiding slice creation!

üìÑ Python Solution (68-5.py) - Most Efficient

import bisect
from typing import List

class Solution:
    def twoSum(self, numbers: List[int], target: int) -> List[int]:
        for k, v in enumerate(numbers):
            expected = target - v
            i = bisect.bisect_left(numbers, expected, k + 1)  # lo=k+1 !
            if i < len(numbers) and numbers[i] == expected:
                return k + 1, i + 1
