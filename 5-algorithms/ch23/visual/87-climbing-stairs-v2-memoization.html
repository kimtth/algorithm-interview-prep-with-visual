Count ways to climb n stairs (1 or 2 steps at a time).

Key Insight: f(n) = f(n-1) + f(n-2). Use memoization (defaultdict) to avoid redundant calculations.

ðŸ“„ Python Solution (87-2.py)

import collections

class Solution:
    dp = collections.defaultdict(int)

    def climbStairs(self, n: int) -> int:
        if n <= 2:
            return n
        if self.dp[n]:
            return self.dp[n]
        self.dp[n] = self.climbStairs(n - 1) + self.climbStairs(n - 2)
        return self.dp[n]
