üìã Problem Description

Find the minimum window in s that contains all characters of t.

Key Insight: Use Python Counter's & (AND) operation! When current_count & t_count == t_count, we have all required characters.

üìÑ Python Solution (76-3.py)

import collections

class Solution:
    def minWindow(self, s: str, t: str) -> str:
        t_count = collections.Counter(t)
        current_count = collections.Counter()
        start = float('-inf')
        end = float('inf')
        left = 0

        for right, char in enumerate(s, 1):
            current_count[char] += 1

            # AND operation: check if we have all required chars
            while current_count & t_count == t_count:
                if right - left < end - start:
                    start, end = left, right
                current_count[s[left]] -= 1
                left += 1

        return s[start: end] if end - start <= len(s) else ''
