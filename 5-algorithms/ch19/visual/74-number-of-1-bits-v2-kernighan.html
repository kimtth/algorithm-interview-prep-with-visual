Count the number of '1' bits in the binary representation of an unsigned integer.

Key Insight: n &= n - 1 removes the rightmost 1-bit. Count iterations until n becomes 0. Only O(k) operations where k is the number of 1-bits!

ðŸ“„ Python Solution (74-2.py)

class Solution:
    def hammingWeight(self, n: int) -> int:
        count = 0
        while n:
            n &= n - 1  # Remove rightmost 1-bit
            count += 1
        return count
