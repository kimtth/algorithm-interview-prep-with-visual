You can climb 1 or 2 steps at a time. How many ways to reach the top?

ğŸ“„ Python Solution (87-1.py)

class Solution:
    def climbStairs(self, n: int) -> int:
        if n == 1:
            return 1
        if n == 2:
            return 2
        return self.climbStairs(n - 1) + self.climbStairs(n - 2)
