Q55: Kth Largest Element (v1 - Max Heap)

Build a max heap using negative values (Python heapq is min heap). Push all elements, then pop k times to get kth largest.

üìÅ Heap ‚è±Ô∏è O(n + k log n) üí° Max Heap via Negation

üíª Python Solution

def findKthLargest(nums: List[int], k: int) -> int:
    heap = list()
    for n in nums:
        heapq.heappush(heap, -n)  # Negate for max heap

    for _ in range(1, k):
        heapq.heappop(heap)

    return -heapq.heappop(heap)
