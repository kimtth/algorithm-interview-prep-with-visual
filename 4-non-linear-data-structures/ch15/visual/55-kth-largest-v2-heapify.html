Q55: Kth Largest Element (v2 - Min Heap)

Heapify array as min heap, then pop (n-k) smallest elements. The top of remaining heap is kth largest.

üìÅ Heap ‚è±Ô∏è O(n + (n-k) log n) üí° Pop Smallest

üíª Python Solution

def findKthLargest(nums: List[int], k: int) -> int:
    heapq.heapify(nums)  # Build min heap in-place

    for _ in range(len(nums) - k):  # Pop n-k times
        heapq.heappop(nums)

    return heapq.heappop(nums)  # Top is kth largest
