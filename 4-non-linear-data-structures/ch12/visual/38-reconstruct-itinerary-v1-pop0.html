Q38: Reconstruct Itinerary - DFS pop(0)

Reconstruct the itinerary from flight tickets. Use all tickets exactly once, starting from "JFK".

üíª Python Solution (DFS with pop(0))

def findItinerary(tickets: List[List[str]]) -> List[str]:
    graph = collections.defaultdict(list)
    # Build graph in sorted order
    for a, b in sorted(tickets):
        graph[a].append(b)

    route = []

    def dfs(a):
        while graph[a]:
            dfs(graph[a].pop(0))  # Visit first (lexical order)
        route.append(a)

    dfs('JFK')
    return route[::-1]  # Reverse for correct order
