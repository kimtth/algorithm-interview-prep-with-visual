Q35: Combinations - DFS

Given n and k, return all possible combinations of k numbers chosen from the range [1, n].

üìÅ Backtracking ‚è±Ô∏è O(C(n,k)) üí° DFS with start index

üíª Python Solution

def combine(n: int, k: int) -> List[List[int]]:
    results = []

    def dfs(elements, start: int, k: int):
        if k == 0:
            results.append(elements[:])
            return

        # Start from current position to avoid duplicates
        for i in range(start, n + 1):
            elements.append(i)
            dfs(elements, i + 1, k - 1)
            elements.pop()  # Backtrack

    dfs([], 1, k)
    return results
