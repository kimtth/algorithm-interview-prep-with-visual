Q36: Combination Sum

Find all unique combinations of candidates that sum to target. Each number can be used unlimited times.

üìÅ Backtracking ‚è±Ô∏è O(n^target) üí° DFS with unlimited reuse

üíª Python Solution

def combinationSum(candidates: List[int], target: int) -> List[List[int]]:
    result = []

    def dfs(csum, index, path):
        if csum < 0:
            return  # Exceeded target, prune
        if csum == 0:
            result.append(path)
            return  # Found valid combination

        # Start from current index (allows reuse of same element)
        for i in range(index, len(candidates)):
            dfs(csum - candidates[i], i, path + [candidates[i]])

    dfs(target, 0, [])
    return result
