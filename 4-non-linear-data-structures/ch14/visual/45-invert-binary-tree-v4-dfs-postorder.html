Q45: Invert Binary Tree - DFS Post-order

Invert using iterative DFS with post-order style: push children first, then swap.

üíª Python Solution

def invertTree(root: TreeNode) -> TreeNode:
    stack = collections.deque([root])
    
    while stack:
        node = stack.pop()
        
        if node:
            # Push children first (before swap)
            stack.append(node.left)
            stack.append(node.right)
            
            # Then swap (post-order style)
            node.left, node.right = node.right, node.left
    
    return root
