Q52: Range Sum of BST (v1 - Brute Force)

Sum all node values in range [L, R]. This version visits ALL nodes recursively without BST-based pruning.

üìÅ BST ‚è±Ô∏è O(n) üí° Recursive DFS

üíª Python Solution

def rangeSumBST(root: TreeNode, L: int, R: int) -> int:
    if not root:
        return 0

    return (root.val if L <= root.val <= R else 0) + \
           rangeSumBST(root.left, L, R) + \
           rangeSumBST(root.right, L, R)
