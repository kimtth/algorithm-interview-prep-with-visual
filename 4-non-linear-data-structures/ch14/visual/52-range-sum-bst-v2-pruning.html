Q52: Range Sum of BST (v2 - Pruning)

Uses BST property to prune branches: if node < L, skip left subtree; if node > R, skip right subtree.

üìÅ BST ‚è±Ô∏è O(n) worst, better avg üí° Recursive with Pruning

üíª Python Solution

def rangeSumBST(root: TreeNode, L: int, R: int) -> int:
    def dfs(node: TreeNode):
        if not node:
            return 0

        if node.val < L:          # Skip left, go right only
            return dfs(node.right)
        elif node.val > R:        # Skip right, go left only
            return dfs(node.left)
        return node.val + dfs(node.left) + dfs(node.right)

    return dfs(root)
