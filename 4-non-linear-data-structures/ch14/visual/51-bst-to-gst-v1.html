Q51: BST to Greater Sum Tree

Convert BST to Greater Sum Tree where each node's value is replaced by the sum of all values greater than or equal to it. Uses reverse in-order traversal (right ‚Üí node ‚Üí left).

class Solution:
    val: int = 0

    def bstToGst(self, root: TreeNode) -> TreeNode:
        # Ï§ëÏúÑ ÏàúÌöå ÎÖ∏Îìú Í∞í ÎàÑÏ†Å (Reverse in-order)
        if root:
            self.bstToGst(root.right)  # Visit right first
            self.val += root.val       # Accumulate
            root.val = self.val        # Update node
            self.bstToGst(root.left)   # Then left

        return root
