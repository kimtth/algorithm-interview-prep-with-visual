Q52: Range Sum of BST (v3 - DFS Stack)

Iterative DFS using stack with BST-based pruning. Push nodes to stack and process with LIFO order.

def rangeSumBST(root: TreeNode, L: int, R: int) -> int:
    stack, sum = [root], 0
    # Ïä§ÌÉù Ïù¥Ïö© ÌïÑÏöîÌïú ÎÖ∏Îìú DFS Î∞òÎ≥µ
    while stack:
        node = stack.pop()
        if node:
            if node.val > L:
                stack.append(node.left)
            if node.val < R:
                stack.append(node.right)
            if L <= node.val <= R:
                sum += node.val
    return sum
