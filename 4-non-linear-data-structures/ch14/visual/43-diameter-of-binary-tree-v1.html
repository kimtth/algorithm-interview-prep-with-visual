Q43: Diameter of Binary Tree

Find the diameter of a binary tree - the longest path between any two nodes. The path may or may not pass through the root.

üíª Python Solution

def diameterOfBinaryTree(root: TreeNode) -> int:
    longest = 0
    
    def dfs(node: TreeNode) -> int:
        nonlocal longest
        if not node:
            return -1
        # DFS to leaf nodes
        left = dfs(node.left)
        right = dfs(node.right)
        
        # Update longest path through this node
        longest = max(longest, left + right + 2)
        # Return height from this node
        return max(left, right) + 1
    
    dfs(root)
    return longest
