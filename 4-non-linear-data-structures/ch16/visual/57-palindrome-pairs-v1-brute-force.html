Q57: Palindrome Pairs (v1 - Brute Force)

Check all pairs of words to find which concatenations form palindromes. O(n¬≤k) where k is word length.

üíª Python Solution

def palindromePairs(words: List[str]) -> List[List[int]]:
    def is_palindrome(word):
        return word == word[::-1]

    output = []
    for i, word1 in enumerate(words):
        for j, word2 in enumerate(words):
            if i == j:
                continue
            if is_palindrome(word1 + word2):
                output.append([i, j])
    return output
