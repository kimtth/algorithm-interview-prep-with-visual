Q17: Swap Nodes in Pairs

Given a linked list, swap every two adjacent nodes and return its head. Actually swap nodes, not values.

üíª Python Solution - Swap Node Pointers

def swapPairs(self, head: ListNode) -> ListNode:
    root = prev = ListNode(None)  # Dummy node
    prev.next = head
    
    while head and head.next:
        b = head.next
        head.next = b.next  # a ‚Üí c
        b.next = head       # b ‚Üí a
        prev.next = b       # prev ‚Üí b
        
        # Move forward
        head = head.next
        prev = prev.next.next
    
    return root.next
