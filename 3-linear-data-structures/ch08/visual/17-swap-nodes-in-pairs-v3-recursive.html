Q17: Swap Nodes in Pairs

Given a linked list, swap every two adjacent nodes and return its head using recursion.

def swapPairs(self, head: ListNode) -> ListNode:
    if head and head.next:
        p = head.next
        # Get swapped result from recursion
        head.next = self.swapPairs(p.next)
        p.next = head
        return p
    return head
