Q8: Trapping Rain Water

Solution 2: Stack Approach

üìã Problem Description

Given n non-negative integers representing elevation map, compute how much water can be trapped.

Idea: Use stack to track indices. When we find a bar taller than stack top, calculate trapped water horizontally.

Time: O(n) Space: O(n)

from typing import List


class Solution:
    def trap(self, height: List[int]) -> int:
        stack = []
        volume = 0

        for i in range(len(height)):
            # Î≥ÄÍ≥°Ï†êÏùÑ ÎßåÎÇòÎäî Í≤ΩÏö∞
            while stack and height[i] > height[stack[-1]]:
                # Ïä§ÌÉùÏóêÏÑú Í∫ºÎÇ∏Îã§
                top = stack.pop()

                if not len(stack):
                    break

                # Ïù¥Ï†ÑÍ≥ºÏùò Ï∞®Ïù¥ÎßåÌÅº Î¨º ÎÜíÏù¥ Ï≤òÎ¶¨
                distance = i - stack[-1] - 1
                waters = min(height[i], height[stack[-1]]) - height[top]

                volume += distance * waters

            stack.append(i)
        return volume
