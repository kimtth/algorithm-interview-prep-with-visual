Q7: Two Sum - Two-Pass Hash Map

Find two numbers that add up to the target. First pass builds a hash map, second pass searches for complements.

üíª Python Solution

def twoSum(nums: List[int], target: int) -> List[int]:
    nums_map = {}
    # First pass: build hash map
    for i, num in enumerate(nums):
        nums_map[num] = i
    
    # Second pass: search for complement
    for i, num in enumerate(nums):
        if target - num in nums_map and nums_map[target - num] != i:
            return [i, nums_map[target - num]]
