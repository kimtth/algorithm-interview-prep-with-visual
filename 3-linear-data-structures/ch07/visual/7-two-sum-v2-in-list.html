Q7: Two Sum - In-List Search

Find two numbers that add up to the target using Python's `in` operator for list searching.

üíª Python Solution

def twoSum(nums: List[int], target: int) -> List[int]:
    for i, n in enumerate(nums):
        complement = target - n
        # Search in remaining slice
        if complement in nums[i + 1:]:
            return [nums.index(n), nums[i + 1:].index(complement) + (i + 1)]
