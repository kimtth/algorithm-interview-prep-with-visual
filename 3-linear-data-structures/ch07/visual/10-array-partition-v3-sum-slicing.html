Given an integer array nums of 2n integers, group these integers into n pairs (a1, b1), (a2, b2), ..., (an, bn) such that the sum of min(ai, bi) for all i is maximized. Return the maximized sum.

Key Insight: After sorting, pair adjacent elements. The smaller one in each pair will be at even indices (0, 2, 4...). Just sum elements at even indices!

ðŸ“„ Python Solution (10-3.py)

from typing import List

class Solution:
    def arrayPairSum(self, nums: List[int]) -> int:
        return sum(sorted(nums)[::2])  # Sum every other element starting from index 0
