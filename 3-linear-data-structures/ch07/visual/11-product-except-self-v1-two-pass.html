Q11: Product of Array Except Self

Given an integer array nums, return an array where each element is the product of all array elements except itself. Solve without division in O(n) time.

üíª Python Solution - Left & Right Sweep

def productExceptSelf(self, nums: List[int]) -> List[int]:
    out = []
    p = 1
    # Left sweep - store products of all left elements
    for i in range(0, len(nums)):
        out.append(p)
        p = p * nums[i]
    
    p = 1
    # Right sweep - multiply by products of all right elements
    for i in range(len(nums) - 1, -1, -1):
        out[i] = out[i] * p
        p = p * nums[i]
    return out
