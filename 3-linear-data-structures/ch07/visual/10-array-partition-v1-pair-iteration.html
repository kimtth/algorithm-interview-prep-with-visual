Q10: Array Partition I

Given an integer array nums of 2n integers, group these integers into n pairs (a‚ÇÅ, b‚ÇÅ), (a‚ÇÇ, b‚ÇÇ), ..., (a‚Çô, b‚Çô) such that the sum of min(a·µ¢, b·µ¢) for all i is maximized.

def arrayPairSum(self, nums: List[int]) -> int:
    sum = 0
    pair = []
    nums.sort()

    for n in nums:
        # Build pairs from front (ascending)
        pair.append(n)
        if len(pair) == 2:
            sum += min(pair)
            pair = []

    return sum
