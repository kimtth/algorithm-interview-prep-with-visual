Q31: Top K Frequent Elements - Pythonic

Given an integer array and k, return the k most frequent elements using Counter.most_common().

üìÅ Counter ‚è±Ô∏è O(n log n) üí° One-liner Pythonic

üíª Python Solution (Pythonic with Counter)

from collections import Counter

def topKFrequent(nums: List[int], k: int) -> List[int]:
    return list(zip(*Counter(nums).most_common(k)))[0]
