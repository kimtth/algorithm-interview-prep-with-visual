Q31: Top K Frequent Elements - Heap

Given an integer array and k, return the k most frequent elements using a min-heap.

üíª Python Solution (Heap with Negative Frequencies)

import heapq
from collections import Counter

def topKFrequent(nums: List[int], k: int) -> List[int]:
    freqs = Counter(nums)
    freqs_heap = []
    # Use negative frequency for max-heap behavior
    for f in freqs:
        heapq.heappush(freqs_heap, (-freqs[f], f))
    
    topk = []
    for _ in range(k):
        topk.append(heapq.heappop(freqs_heap)[1])
    return topk
