Q4: Most Common Word

Regex + Counter

Given a paragraph and a list of banned words, find the most frequent word that is not in the banned list. It is guaranteed there is at least one word that isn't banned.

Input: paragraph = "Bob hit a ball, the hit BALL flew far after it was hit."
banned = ["hit"]

Time: O(n) Space: O(n)

üíª Python Code (4-1.py)

import collections
import re
from typing import List


class Solution:
    def mostCommonWord(self, paragraph: str, 
                         banned: List[str]) -> str:
        words = [word for word in 
            re.sub(r'[^\w]', ' ', paragraph)
            .lower().split()
                 if word not in banned]

        counts = collections.Counter(words)
        # Í∞ÄÏû• ÌùîÌïòÍ≤å Îì±Ïû•ÌïòÎäî Îã®Ïñ¥Ïùò Ï≤´ Î≤àÏß∏ Ïù∏Îç±Ïä§
        return counts.most_common(1)[0][0]
